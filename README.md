# NEA Project Overview
This project is a comprehensive 2D arena combat game developed as an A-Level Computer Science Non-Examined Assessment (NEA) using C# and the MonoGame framework. It features both local PvP multiplayer and a single-player training mode where players can fight against varying difficulties of AI opponents. Players select from distinct combat classes such as Mage, Melee, Ranger, and Summoner to wield quirky, light-hearted weapons across creatively themed maps. Under the hood, the games achieves high technical complexity by implementing a custom finite state machine for scene management, bespoke data structures such as linked lists, stacks, and circular queues, and integrated SQL database support for persistent user accounts, a progression-based upgrade shop, and competitive leaderboards.

This directory reveals several high-level programming techniques specifically implemented to score highly on the NEA rubric (A* grade):
- Custom Data Structures
- Object-Oriented Programming, including heavy use of inheritance and polymorphism, aswell as abstraction for problem solving to improve code modularity
- Artificial Intelligence
- Database Integration
