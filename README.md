# Overview
This repo contains a comprehensive collection of C# programming projects, coursework, and coding challenges. It demonstrates a progression of programming skills, ranging from fundamental console applications and algorithm design, to GUIs and database integrations. 

# Structure
The portfolio is organised into key learning modules and project categories:
- Algorithm Design
- Arrays
- Assessments & Mocks
- Coding Challenges (OCR)
- Data Representation
- Exception Handling
- Graphical Work
- File Handling

# Technologies and Frameworks
- Primary Language: C#
- Application Types: Console Applications and Windows GUI Applications
- Data Management: SQLite Integration for persistent data storage

# NEA Project Overview
This directory reveals several high-level programming techniques specifically implemented to score highly on the NEA rubric (A* grade):
- Custom Data Structures
- Object-Oriented Programming, including heavy use of inheritance and polymorphism, aswell as abstraction for problem solving to improve code modularity
- Artificial Intelligence
- Database Integration

# Instructions
If you would like to explore any of these projects in more detail, follow these instructions:
1. Navigate to the folder of the specific project you wish to view,
2. Open the corresponding ```.sln``` file in your preferred IDE.
3. Each project is self-contained within its own ```.csproj``` configuration, allowing them to be built and run independently of each other.
